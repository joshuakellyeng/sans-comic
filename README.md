# Sans Comic

An e-commerce web application that allows customers to purchase digital comics and graphic novels. Built with JavaScript, React, Tailwind CSS, and DaisyUI. [Live Link](https://sanscomic.netlify.app/)<br/>

## Technologies Used

- JavaScript
